Contents
  • vmalloc/vfree
    • vmalloc will return a virtual address of a chunk of memory.
    • Memory obtained with vmalloc is not necessarily physically contiguous, the pages may be all over the place.
    • Memory obtained with vmalloc is not (at least not easily) DMA-able
    • Use vmalloc when you need memory for a purely software construct which does not need to be touched by any device.
